# AI Ops Profile (SEO Optimized)

This repository contains an SEO-friendly static profile page for a professional working in GenAI, AI Agents, MLOps, AIOps, DevOps, and Cloud DevOps.

## What was added for SEO
- Semantic HTML structure (`header`, `main`, `section`, `footer`, accessible navigation).
- Search metadata (`description`, `keywords`, `robots`, canonical URL).
- Open Graph and Twitter cards for better social previews.
- Structured data (JSON-LD `Person` schema) for richer search understanding.
- `robots.txt` and `sitemap.xml` for crawler guidance.
- Better accessibility and performance defaults (image dimensions, lazy-loading, improved alt text).

## Usage
1. Clone this repository.
2. Update all placeholders in `index.html`:
   - `YOUR_GITHUB_USERNAME`
   - `YOUR_LINKEDIN`
   - `YOUR_EMAIL@example.com`
3. If hosted on GitHub Pages, keep canonical and sitemap URLs aligned with your actual URL.
4. Push changes and submit your website URL in Google Search Console for faster indexing.

## Production checklist
- Replace placeholders with real identities and links.
- Add real project links.
- Keep content updated regularly.
- Validate structured data in Google's Rich Results Test.
- Verify indexing status in Google Search Console.
